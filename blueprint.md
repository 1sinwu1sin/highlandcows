# Kwon Do-hoo's Learning Coaching Homepage Blueprint

## Overview

A simple homepage introducing "권도후의 학습코칭" (Kwon Do-hoo's Learning Coaching). The page will provide a brief overview of the coaching, contact information, and a call to action.

## Project Structure

- `index.html`: The main HTML file for the homepage.
- `style.css`: The CSS file for styling the homepage.
- `main.js`: (Optional) JavaScript file for any interactivity.
- `blueprint.md`: This file, documenting the project.
